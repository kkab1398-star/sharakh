# قائمة الاختبار - ميزة مشاركة بيانات السائق عبر واتساب

## ✅ نقاط الاختبار:

### 1. إضافة سائق جديد
- [ ] اذهب إلى `/dashboard/drivers/new`
- [ ] ملء البيانات:
  - [ ] الاسم الكامل: محمد أحمد
  - [ ] اسم المستخدم: ahmed123
  - [ ] كلمة المرور: 123456
  - [ ] رقم الجوال: 0501234567 أو +966501234567
  - [ ] نسبة الأرباح: 50%
- [ ] اضغط "إضافة السائق"
- [ ] تحقق من ظهور الـ modal الاحتفالي

### 2. التحقق من محتوى الـ Modal عند الإضافة الجديدة
- [ ] الرسالة الخضراء: "✅ تم إضافة السائق بنجاح"
- [ ] اسم السائق ظاهر تحت الرسالة
- [ ] بطاقة الرسالة تحتوي على:
  - [ ] رسالة الترحيب
  - [ ] رابط الدخول
  - [ ] اسم المستخدم
  - [ ] كلمة المرور
  - [ ] تحذير الأمان

### 3. زر "فتح واتساب"
- [ ] يظهر الزر باللون الأخضر (#25D366)
- [ ] اضغط عليه - يجب أن يفتح واتساب في تبويب جديد
- [ ] الرسالة تحتوي على:
  - [ ] رقم الجوال من النموذج (محوّل إلى صيغة دولية)
  - [ ] محتوى الرسالة كاملاً مع جميع البيانات

### 4. زر "نسخ الرسالة"
- [ ] يظهر الزر باللون الأصفر (#FFCD11)
- [ ] اضغط عليه - يجب أن يتغير نصه إلى "✓ تم النسخ"
- [ ] يعود الاسم الأصلي بعد ثانيتين
- [ ] الرسالة تُنسخ للحافظة بالفعل

### 5. زر "إغلاق"
- [ ] يظهر الزر باللون الرمادي (#3D3D3D)
- [ ] اضغط عليه - يجب أن يغلق الـ modal
- [ ] ينقلك لصفحة قائمة السائقين

### 6. صفحة تفاصيل السائق
- [ ] اذهب إلى قائمة السائقين `/dashboard/drivers`
- [ ] اضغط على "تفاصيل" لأحد السائقين
- [ ] تحقق من وجود الأزرار الجديدة:
  - [ ] 📱 إعادة إرسال بيانات الدخول (أصفر)
  - [ ] 🔑 إعادة تعيين كلمة المرور (أزرق)

### 7. اختبار "إعادة إرسال بيانات الدخول"
- [ ] اضغط على الزر
- [ ] يظهر modal بنفس التصميم لكن:
  - [ ] العنوان: "إعادة إرسال بيانات الدخول"
  - [ ] **بدون كلمة المرور** في الرسالة
  - [ ] الرسالة تحتوي فقط على:
    - [ ] رابط الدخول
    - [ ] اسم المستخدم
    - [ ] التحذير الأمني
- [ ] اختبر أزرار النسخ والواتساب

### 8. اختبار "إعادة تعيين كلمة المرور"
- [ ] اضغط على الزر
- [ ] يظهر modal جديد بعنوان "إعادة تعيين كلمة المرور"
- [ ] يحتوي على:
  - [ ] حقل "كلمة المرور الجديدة"
  - [ ] حقل "تأكيد كلمة المرور"
  - [ ] زر لإظهار/إخفاء كلمة المرور (👁️)
  - [ ] زر "تحديث" (أصفر)
  - [ ] زر "إلغاء" (رمادي)

### 9. تحديث كلمة المرور
- [ ] أدخل كلمة مرور جديدة: `newpass123`
- [ ] أكد كلمة المرور
- [ ] اضغط "تحديث"
- [ ] يجب أن تنجح العملية برسالة "تم تحديث كلمة المرور بنجاح"
- [ ] يعود الـ modal لـ "إعادة إرسال بيانات الدخول"
- [ ] الرسالة الآن تحتوي على:
  - [ ] كلمة المرور الجديدة `newpass123`
  - [ ] رابط الدخول
  - [ ] اسم المستخدم

### 10. حالات الخطأ
- [ ] جرب إدخال كلمة مرور أقل من 6 أحرف - يجب ظهور خطأ
- [ ] جرب عدم تطابق كلمات المرور - يجب ظهور خطأ
- [ ] جرب بدون حقول - يجب تعطيل الزر

### 11. رابط الدخول
- [ ] تحقق من أن رابط الدخول يحتوي على:
  - [ ] إذا كان للشريك slug: `[APP_URL]/login/[slug]`
  - [ ] إذا لم يكن: `[APP_URL]/driver/login`

### 12. صيغة رقم الجوال
- [ ] اختبر الأرقام المختلفة:
  - [ ] `05xxxxxxxx` → يتحول إلى `966 5xxxxxxxx`
  - [ ] `+9665xxxxxxxx` → يبقى كما هو
  - [ ] `9665xxxxxxxx` → يبقى كما هو
