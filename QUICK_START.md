# 🚀 دليل البدء السريع - اختبار ميزة مشاركة السائق عبر واتساب

## قبل البدء:
```bash
npm run dev
# الخادم يعمل على http://localhost:3000
```

---

## 🎯 اختبار سريع (5 دقائق):

### الخطوة 1️⃣: إضافة سائق جديد
```
1. انتقل إلى: http://localhost:3000/dashboard/drivers/new
2. ملء البيانات:
   • الاسم الكامل: محمد أحمد
   • اسم المستخدم: ahmed_123
   • كلمة المرور: password123
   • رقم الجوال: 0501234567
   • نسبة الأرباح: 50%
3. اضغط "إضافة السائق"
```

### الخطوة 2️⃣: اختبار الـ Modal
```
بعد الحفظ مباشرة:
✅ سيظهر modal احتفالي
✅ عنوان أخضر: "تم إضافة السائق بنجاح"
✅ اسم السائق ظاهر تحته
✅ بطاقة الرسالة تحتوي على:
   - رسالة الترحيب
   - رابط الدخول
   - اسم المستخدم
   - كلمة المرور
```

### الخطوة 3️⃣: اختبار الأزرار
```
في الـ Modal:

1️⃣ اضغط "💬 فتح واتساب"
   ✅ يجب أن يفتح واتساب في تبويب جديد
   ✅ الرسالة جاهزة للإرسال

2️⃣ اضغط "📋 نسخ الرسالة"
   ✅ النص يتغير إلى "✓ تم النسخ"
   ✅ يعود بعد ثانيتين

3️⃣ اضغط "إغلاق"
   ✅ ينغلق الـ modal
   ✅ ينقلك لقائمة السائقين
```

### الخطوة 4️⃣: اختبار صفحة التفاصيل
```
1. من قائمة السائقين: /dashboard/drivers
2. اضغط "تفاصيل" على السائق الذي أنشأته
3. يجب أن ترى زرين جديدين:
   • 📱 إعادة إرسال بيانات الدخول
   • 🔑 إعادة تعيين كلمة المرور
```

### الخطوة 5️⃣: اختبار إعادة الإرسال
```
اضغط على "📱 إعادة إرسال بيانات الدخول":

✅ يظهر modal بنفس التصميم
✅ العنوان: "إعادة إرسال بيانات الدخول"
✅ **بدون كلمة المرور** في الرسالة
✅ الرسالة تحتوي على:
   - رابط الدخول
   - اسم المستخدم فقط
```

### الخطوة 6️⃣: اختبار تغيير كلمة المرور
```
اضغط على "🔑 إعادة تعيين كلمة المرور":

✅ يظهر modal جديد
✅ يحتوي على:
   • حقل كلمة المرور الجديدة
   • حقل تأكيد كلمة المرور
   • زر إظهار/إخفاء (👁️)

الخطوات:
1. أدخل كلمة مرور جديدة: newpass123
2. أكد نفس كلمة المرور
3. اضغط "تحديث"
4. يجب ظهور رسالة النجاح
5. الـ modal الجديد سيظهر مع كلمة المرور الجديدة
```
