# وثيقة تنفيذ ميزة مشاركة بيانات السائق عبر واتساب

## الملفات المنشأة/المحدثة:

### 1. API Endpoint جديد
- **`app/api/drivers/[id]/password/route.ts`**
  - طريقة: `PUT /api/drivers/:id/password`
  - الوظيفة: تحديث كلمة المرور للسائق
  - المتطلبات: تحديث كلمة المرور للسائق المسجل

### 2. Components جديد
- **`components/ResetPasswordModal.tsx`**
  - Modal لإعادة تعيين كلمة المرور للسائق
  - يحتوي على:
    - حقل كلمة المرور الجديدة
    - حقل تأكيد كلمة المرور
    - زر لإظهار/إخفاء كلمة المرور
    - عند النجاح، يعود الـ modal القديم لإعادة إرسال الرسالة مع كلمة المرور الجديدة

### 3. Components محدثة
- **`components/ShareDriverModal.tsx`**
  - تم إضافة prop جديد: `showPassword` (قيمة افتراضية: `true`)
  - يدعم الآن:
    - وضع "إضافة سائق جديد" (مع كلمة المرور)
    - وضع "إعادة إرسال بيانات الدخول" (بدون كلمة المرور)
  - النص العنوان يتغير حسب الوضع

- **`app/dashboard/drivers/new/page.tsx`**
  - بالفعل كانت تستخدم `ShareDriverModal` بشكل صحيح
  - تم تحديث الـ import للإشارة للـ component بدلاً من استيراده

- **`app/dashboard/drivers/[id]/page.tsx`** (صفحة تفاصيل السائق)
  - أضيف زر "إعادة إرسال بيانات الدخول" 📱
  - أضيف زر "إعادة تعيين كلمة المرور" 🔑
  - أضيفت حالات جديدة (states):
    - `showShareModal`: للتحكم في ظهور modal المشاركة
    - `showResetModal`: للتحكم في ظهور modal تغيير كلمة المرور
    - `newPassword`: لتخزين كلمة المرور الجديدة بعد التحديث

## سير العمل:

### عند إضافة سائق جديد:
1. ملء البيانات وحفظ السائق
2. ظهور modal احتفالي بـ:
   - رسالة نجاح: "✅ تم إضافة السائق بنجاح"
   - بطاقة رسالة واتساب تحتوي على:
     - رابط الدخول
     - اسم المستخدم
     - كلمة المرور
   - زر "فتح واتساب" (أخضر #25D366)
   - زر "نسخ الرسالة" (أصفر)
   - زر "إغلاق" (رمادي)

### إعادة إرسال بيانات الدخول:
1. في صفحة تفاصيل السائق، اضغط على "📱 إعادة إرسال بيانات الدخول"
2. يظهر modal بنفس التصميم لكن بدون كلمة المرور
3. الرسالة تحتوي على:
   - رابط الدخول
   - اسم المستخدم فقط
   - رسالة تحذيرية

### إعادة تعيين كلمة المرور:
1. في صفحة تفاصيل السائق، اضغط على "🔑 إعادة تعيين كلمة المرور"
2. يظهر modal بسيط يحتوي على:
   - حقل كلمة المرور الجديدة
   - حقل تأكيد كلمة المرور
   - زر لإظهار/إخفاء كلمة المرور (👁️)
   - عند النجاح:
     - رسالة نجاح
     - يمكن فوراً اضغط "إعادة إرسال بيانات الدخول" لإرسالها مع كلمة المرور الجديدة

## رابط السائق:

يتم بناء رابط الدخول كالتالي:
- إذا كان للشريك `slug`: `[APP_URL]/login/[slug]`
- إذا لم يكن: `[APP_URL]/driver/login`

من الـ `.env.local`:
```
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## معايير التصميم:

### Modal الرئيسي (ShareDriverModal):
- الخلفية: `#1A1A1A`
- الإطار: `2px solid #FFCD11`
- border-radius: `8px`

### بطاقة الرسالة:
- الخلفية: `#2A2A2A`
- الخط: `monospace`

### الأزرار:
- واتساب: أخضر `#25D366`
- نسخ: أصفر `#FFCD11` بإطار أصفر
- إغلاق: رمادي `#3D3D3D`

## الاختبار:

لاختبار الوظيفة:
1. اذهب إلى `/dashboard/drivers/new`
2. أضف سائق جديد مع رقم جوال (05XXXXXXXX أو +966XXXXXXXXX)
3. بعد الحفظ، يجب أن يظهر الـ modal الاحتفالي
4. اضغط "فتح واتساب" - يجب أن يفتح واتساب مع الرسالة الجاهزة
5. اضغط "نسخ الرسالة" - يجب أن تُنسخ الرسالة للحافظة
6. اذهب لتفاصيل السائق
7. اختبر "إعادة إرسال بيانات الدخول"
8. اختبر "إعادة تعيين كلمة المرور" وتحديث كلمة المرور الجديدة
